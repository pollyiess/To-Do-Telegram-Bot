# To-Do Telegram Bot

Простой и эффективный менеджер задач на Ruby с поддержкой состояний и базой данных SQLite.

## Функционал
- Добавление задач с выбором приоритета;
- Просмотр списка текущих дел с сортировкой по важности;
- Отметка задач как выполненных через кнопки;
- Очистка истории выполненных задач.
