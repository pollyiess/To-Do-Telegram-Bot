# To-Do Telegram Bot

Простой и эффективный Telegram-бот для управления списком задач с системой приоритетов.

## Функционал
- Добавление задач с выбором приоритета;
- Просмотр списка текущих дел с сортировкой по важности;
- Очистка списка задач.

## Инструкция по запуску
#### 1. Склонируйте репозиторий
#### 2. Установите зависимости:
bundle install
#### 3. Создайте бота через @BotFather, вставьте токен в окружение своего терминала:
$env:TELEGRAM_BOT_TOKEN="ваш_токен_здесь"
#### 4. Запустите бота:
ruby bin/bot.rb
